## 一、算法基础

涵盖经典算法范式与基础算法实现，是计算机科学的核心内容。

- [sorts](https://github.com/TheAlgorithms/Python/tree/master/sorts) — 排序算法（冒泡排序、快速排序、归并排序、堆排序、计数排序、基数排序、桶排序、希尔排序、 Tim Sort 等）
- [searches](https://github.com/TheAlgorithms/Python/tree/master/searches) — 搜索算法（二分查找、线性搜索、插值搜索、跳跃搜索、指数搜索、斐波那契搜索、三元搜索等）
- [graphs](https://github.com/TheAlgorithms/Python/tree/master/graphs) — 图论算法（BFS、DFS、Dijkstra 最短路径、Bellman-Ford、Floyd-Warshall、最小生成树、拓扑排序、强连通分量等）
- [dynamic_programming](https://github.com/TheAlgorithms/Python/tree/master/dynamic_programming) — 动态规划（0-1 背包、最长公共子序列、最长递增子序列、编辑距离、斐波那契数列、硬币找零、矩阵链乘法等）
- [greedy_methods](https://github.com/TheAlgorithms/Python/tree/master/greedy_methods) — 贪心算法（活动选择问题、霍夫曼编码、分数背包、任务调度、最小生成树 Prim/Kruskal 等）
- [divide_and_conquer](https://github.com/TheAlgorithms/Python/tree/master/divide_and_conquer) — 分治算法（归并排序、快速排序、最近点对问题、Strassen 矩阵乘法、幂运算等）
- [backtracking](https://github.com/TheAlgorithms/Python/tree/master/backtracking) — 回溯算法（N 皇后问题、数独求解器、迷宫求解、子集和问题、排列组合、图着色等）
- [strings](https://github.com/TheAlgorithms/Python/tree/master/strings) — 字符串算法（KMP 匹配、Rabin-Karp、Manacher 算法、最长回文子串、字符串匹配、正则表达式等）

## 使用提示

### 如何浏览
1. 点击上述任意目录链接，即可跳转到 GitHub 仓库对应目录浏览源码。
2. 每个目录内通常包含多个独立 `.py` 文件，文件名即算法名，可直接阅读。
3. 部分文件底部包含 `__main__` 测试块，可复制到本地直接运行验证。

### 如何贡献
1. Fork [TheAlgorithms/Python](https://github.com/TheAlgorithms/Python) 仓库。
2. 在对应分类目录下添加新的算法实现，确保代码风格与现有文件一致。
3. 为新算法编写清晰的文档字符串（docstring）和类型注解。
4. 提交 Pull Request，遵循项目的 [CONTRIBUTING.md](https://github.com/TheAlgorithms/Python/blob/master/CONTRIBUTING.md) 指南。

本项目采用 [MIT License](https://github.com/TheAlgorithms/Python/blob/master/LICENSE.md) 开源协议，可自由学习、使用和二次开发。
